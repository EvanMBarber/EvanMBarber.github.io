Variables and Data Types

Learn how to store, update, and work with data using JavaScript variables, strings, numbers, and booleans.

In JavaScript, variables are used to store and manage data that your program can use and modify. You can declare variables using let, const, or var — though modern JavaScript mainly uses let (for values that can change) and const (for values that stay constant). For example,

  • String – text enclosed in quotes, e.g. "Hello", 'World', or `Template Literal`.
  • Number – any numeric value, including decimals and negatives, e.g. 42 or 3.14.
  • Boolean – represents truth values: true or false.
  • Undefined – a variable declared but not given a value yet, e.g. let name; results in undefined.
  • Null – an intentional “empty” value, e.g. let result = null;.
  • Symbol – a unique identifier, often used for special object properties.
  • BigInt – used for very large integers beyond the normal numeric limit.

    

  


  

    
Functions and Loops

    
Understand how to make your code dynamic with functions and repetitive with loops.

    
A function is a reusable block of code that performs a specific task. You define a function using the function keyword or newer syntax like arrow functions (=>). Functions make code cleaner and easier to manage by avoiding repetition. You can pass values to functions as parameters and return results using the return statement.

    

      
      
      

    

  


  

    
DOM Manipulation

    
Discover how JavaScript can dynamically change the content and style of web pages through the DOM.

    

  The Document Object Model (DOM) is how JavaScript interacts with the structure of a web page.
  When a webpage loads, the browser turns the HTML into a tree-like structure of objects — each element
  (like <p>, <div>, or <button>)
  becomes a node in this tree. JavaScript can use the DOM to read, modify, add, or remove elements
  dynamically, allowing you to make your pages interactive and responsive.
